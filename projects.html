MindCore – Emotionally Intelligent AI Companion

Project Overview

MindCore is an emotionally intelligent AI companion chatbot that I've been developing since January 2023. It integrates custom Natural Language Understanding (NLU) models with advanced Large Language Models to create truly personalized interactions.

Key Achievement: 98% accuracy in emotional state detection

The system uses a fine-tuned RoBERTa-v3 model for emotional context recognition, creating natural conversations that adapt to the user's evolving emotional states.

Technical Implementation

The core architecture includes:

  • Dynamic Memory System: Retains and contextualizes user conversations, preferences, and life events
  • Emotional Intelligence: Custom NLU models with 98% accuracy in emotional state detection
  • Proactive Engagement: Check-in functionality that initiates conversations based on user patterns
  • Tech Stack: Python, TensorFlow, fine-tuned RoBERTa-v3 model, custom NLP pipeline

Innovation & Impact

What makes MindCore unique:

  • Goes beyond simple chatbot responses to understand and remember emotional context
  • Learns from each interaction to provide increasingly personalized support
  • Proactively reaches out to users based on their emotional patterns and needs
  • Combines cutting-edge AI research with practical emotional intelligence applications

Ongoing Development

This project represents my long-term commitment to advancing AI that truly understands and supports human emotional well-being. It showcases my ability to work on complex, multi-year technical projects while staying current with the latest AI/ML research.

SongDNA – AI Music Analysis Platform (Acquired)

Project Overview

SongDNA was an AI-powered music analysis platform I built from May to September 2025. The platform could extract over 600,000 structured data points per track, enabling advanced song similarity discovery and personalized recommendations that went far beyond what Spotify or Apple Music APIs could provide.

Successfully Acquired by Ben Weston Music in September 2025

The platform processed hundreds of songs into a searchable database, improving music discovery efficiency by 50% compared to baseline metadata systems.

Technical Architecture

The platform was built with a focus on scalability and performance:

  • Data Pipeline: Python-based audio analysis extracting 600K+ data points per track
  • Frontend: TypeScript-based interface for music discovery and recommendations
  • Database: PostgreSQL for structured music data storage and complex queries
  • API Design: REST API enabling third-party integrations and data access

Key Features & Innovation

  • Deep Audio Analysis: Extracted far more data points than traditional music platforms
  • Advanced Similarity Matching: AI algorithms for discovering musically similar tracks
  • Personalized Recommendations: Machine learning-driven suggestion engine
  • Scalable Processing: Handled hundreds of songs with efficient batch processing

Business Impact

The successful acquisition demonstrates:

  • My ability to build market-ready products that solve real problems
  • Technical execution that impressed industry professionals
  • Understanding of both the technical and business aspects of software development
  • Capacity to deliver projects from concept to successful exit
